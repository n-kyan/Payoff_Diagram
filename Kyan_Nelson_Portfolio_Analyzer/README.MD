Kyan Nelson

## SETUP (One-time):
### Windows:
1. Install Python 3.13 from python.org
2. Right-click this folder → "Open in Terminal" (Windows 11) or hold Shift + right-click → "Open PowerShell window here"
3. Type: `pip install -r requirements.txt` and press Enter

### Mac:
1. Install Python 3.13 from python.org
2. Right-click this folder → "New Terminal at Folder"
3. Type: `pip3 install -r requirements.txt` and press Enter

## TO RUN:

### Windows:
1. Right-click this folder → "Open in Terminal"
2. Type: `streamlit run app.py` and press Enter
3. Ignore any prompts from streamlit for email and press enter
4. Browser opens automatically

### Mac:
1. Right-click this folder → "New Terminal at Folder"
2. Type: `streamlit run app.py` and press Enter
3. Ignore any prompts from streamlit for email and press enter
4. Browser opens automatically

The application includes:
- Portfolio Input page
    - Options
    - Debt
    - Forwards
